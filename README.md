# Daletou Analysis（大乐透选号分析）

> **超级大乐透（体彩大乐透）标准 5 步分析流程 + 最大覆盖优化引擎**
>
> **仅供娱乐参考，不构成任何购彩建议。** ⚠️

---

## 概述（Overview）

本项目将「超级大乐透选号分析」固化为一套 **可复现、可审计、离线友好** 的标准流程，把频率学、贝叶斯、信息熵、组合覆盖设计与蒙特卡洛模拟等数学模型落到可直接运行的 Python 脚本上。

核心逻辑：将近 20 期 + 上期开奖数据，按固定 5 步流程做成「冷热清单 / 三区校验 / 后区搭配 / 选号约束 / 成品号码」的可复现分析报告；再叠加一个「最大覆盖优化」引擎，用蒙特卡洛诚实量化它能（和不能）做什么。

## 作用（Purpose）

| 功能 | 说明 |
|------|------|
| 数据存档 | 近 20 期 + 上期原始数据，拆分前/后区，输出冷热号码清单 |
| 前区分区 | 一区(01-10)/二区(11-20)/三区(21-35) 三区全覆盖校验 |
| 后区搭配 | 后区冷热划分 + 两码和值锁定 9-16 + 禁止全冷/全热 |
| 选号约束 | 连号分析 + 23区间胆码 + 防扎堆单区 |
| 成品输出 | 5 组标准"5前2后"号码 + 每组冷热/分区/备注 |
| 覆盖优化 | 最大覆盖贪心算法 + λ=0.03历史热度权重 |
| 蒙特卡洛复查 | N=40万样本 + 95%置信区间 + z检验 |

## 机制（Mechanism）

### 标准 5 步流程（强制顺序）

```
第1步 数据存档 → 第2步 前区分区 → 第3步 后区搭配 → 第4步 选号约束 → 第5步 成品输出
       ↓                                                              ↓
  dlt_raw_21.csv（单一数据源）                              5组"5前2后"+备注
```

### 优化层（可选第 6 层）

`dlt_maxcover.py` 在 5 步硬约束之上，用**最大覆盖贪心算法**选 5 注，使 5 注在 35+12 号码空间铺得最散。

**数学诚实性**：
- ✅ **可优化**：5 注越分散 → 「至少命中任一阵容（含末等奖）」联合概率越高（蒙特卡洛 ~31% vs 随机 29%）
- ❌ **不可优化**：一等奖概率恒 **1/21,425,712**（C(35,5)×C(12,2)），任何策略零影响

## 算法（Algorithms）

| 模块 | 数学方法 | 关键参数 | 输出 |
|------|---------|---------|------|
| 冷热分类 | 频率统计 + 均值±0.5std | 热阈值≥均值+0.5std | 前/后区热/温/冷标签+遗漏 |
| 三区校验 | 分区枚举 + 覆盖判定 | 一区01-10/二区11-20/三区21-35 | 每期三区满足率(~71%) |
| 后区规则 | 和值约束 + 冷热互斥 | 和值∈[9,16],禁冷冷/热热 | 合法后区搭配池 |
| 选号约束 | 连号检测 + 胆码筛选 | 23区间=二区+三区(25码) | 强胆列表+防扎堆判定 |
| 覆盖优化 | 贪心覆盖 + 热度权重λ=0.03 | 前区去重/35, 后区/12 | 优化版5组(+0.33pp) |
| 复查验证 | 蒙特卡洛 N=400,000 | 95%CI + 两比例z检验 | 防结论噪声夸大 |

## 口径铁律（自定义约束，官方无此规定）

- **前区三区**：一区 01-10 ｜ 二区 11-20 ｜ 三区 21-35
- **后区冷热**（基于 21 期）：热 [02,11,12] ｜ 冷 [03,06,09] ｜ 温 [01,04,05,07,08,10]
- **后区和值** ∈ [9,16]，禁止 冷冷 / 热热
- **23 区间胆码池** = 二区+三区共 25 码（一区不纳入）
- **强胆**（≥4次）：13 / 26 / 15 / 19 / 21 / 12 / 20 / 22 / 29
- **防扎堆**：任一区 ≤3，严禁单区 ≥4

> 回测显示历史仅 ~71%（三区）/ 52%（后区规则）满足这些约束——这是预期内的「过滤」，不是「预测」。

## Quick Start

```bash
mkdir dlt-work && cd dlt-work
# 下载本仓库脚本到 scripts/
git clone https://github.com/TouKenAI/daletou-analysis.git

python scripts/analyze.py        # 1.数据存档+冷热
python scripts/zone_check.py     # 2.前区分区校验
python scripts/back_rule.py      # 3.后区冷热搭配
python scripts/select_ref.py     # 4.选号参考约束
python scripts/final_numbers.py  # 5.成品号码（标准5组）

# 可选：优化 + 复查
python scripts/dlt_maxcover.py   # 最大覆盖优化5组
python scripts/dlt_review.py     # 蒙特卡洛复查
```

所有脚本仅依赖 Python 标准库，无需安装第三方包。
